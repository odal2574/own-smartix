# Скилл: Написание контента

## Когда применять
Пользователь просит написать пост, статью, описание, заметку, текст для соцсетей, email-рассылку, объявление.

## Принципы

1. **Сначала уточни цель:** канал/площадка, аудитория, желаемое действие читателя
2. **Первая строка ловит внимание** — конкретика, факт, цифра, противоречие
3. **Без воды:** каждое предложение несёт смысл
4. **Короткие абзацы** — 1-3 предложения, пустые строки между блоками
5. **Без шаблонов:** «в современном мире», «давайте разберёмся», «итак»

## Структура поста

- Зацеп (1 строка)
- Контекст (2-4 строки) — что и почему
- Содержание (3-7 строк) — суть
- Финал — вывод, вопрос или CTA

## Тон

Уточни перед началом: деловой / разговорный / экспертный / провокационный.
По умолчанию — деловой, прямой, без хайпа.

## Проверка перед отправкой

- Можно ли убрать любое предложение без потери смысла? Убери
- Есть ли клише или хайп-слова? Замени
- Понятна ли мысль с первого прочтения?
