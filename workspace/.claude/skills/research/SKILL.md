# Скилл: Исследование и анализ

## Когда применять
Пользователь просит изучить тему, найти данные, сравнить варианты, подобрать конкурентов, проверить факт.

## Алгоритм

1. **Уточни цель** — для чего исследование (статья, решение, презентация, КП)
2. **Определи источники** — официальные сайты, документация, статистика, отраслевые отчёты
3. **Собери данные** — конкретные цифры, цитаты, ссылки
4. **Структурируй** — таблица или список с источниками
5. **Сделай вывод** — что из этого следует для пользователя

## Формат отчёта

```
# Тема: [запрос]

## Что нашёл
- Факт 1 (источник)
- Факт 2 (источник)

## Сравнение / Таблица
[если применимо]

## Вывод
1-3 строки: что это значит для пользователя

## Источники
- [Название](URL)
```

## Принципы

- **Свежесть важна:** если данные старше 2 лет — ищи свежее
- **Проверка фактов:** один источник — гипотеза, два — факт
- **Без выдумок:** не знаешь — скажи «не нашёл», предложи где искать дальше
- **Конкретика:** «выручка $3.2 млрд за 2025» лучше чем «крупная компания»
